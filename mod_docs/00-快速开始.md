# Mod 开发快速开始

## 目录结构

```
res://mods/你的Mod名字/
├── mod.json                 # 必需：Mod 清单
├── scripts/
│   └── main.gd              # 脚本 Mod（type: "script"）
└── data/
    └── balance.json         # 数据 Mod（type: "data"）
```

## mod.json 格式

```json
{
  "name": "我的 Mod",
  "version": "1.0",
  "author": "你的名字",
  "description": "功能介绍",
  "type": "script",
  "min_game_version": "0.1",
  "dependencies": ["另一个Mod"],
  "optional_dependencies": ["可选增强"],
  "conflicts": ["老版本Mod"]
}
```

| 字段 | 说明 |
|------|------|
| `type` | `"script"`（GDScript）/ `"data"`（数据）/ `"language"`（语言） |
| `dependencies` | 硬依赖，缺少则跳过加载 |
| `optional_dependencies` | 可选依赖，有就排前面加载，没有不影响 |
| `conflicts` | 冲突列表，同时启用时报错 |

## 脚本 Mod（推荐）

```gdscript
extends Node

func OnLoad(gm, bridge):
	bridge.log("Hello!")
	bridge.add_money(100000)
```

- `gm` = GameManager（游戏主管理器）
- `bridge` = ModBridge（**所有 C# API 的蛇形封装，主要用这个**）
- 直接写 `bridge.xxx()` 即可调用各种功能

## 数据 Mod

在 `data/balance.json` 中覆盖游戏参数，无需写代码：

```json
{
  "project.init.gameplay_base": 40,
  "fatigue.monthly_gain": 2,
  "sales.multiplier.95": 2.0
}
```

## 工作示例

`mod_examples/` 目录下有可直接运行的示例：

| 示例 | 类型 | 功能 |
|------|------|------|
| `01_money_helper` | 脚本 | 每月加钱 + 按 0 键加钱 |
| `02_dev_booster` | 脚本 | F5 满分 / F6 清疲劳 / F7 全科技 |
| `03_balance_tweaker` | 数据 | 调整初始属性和销售参数 |

## 快速测试

1. 把 `mod_examples/01_money_helper` 复制到 `mods/` 下
2. 启动游戏 → 主菜单 → Mod → 勾选 Money Helper
3. 点"新游戏"进入游戏
4. 按键盘 **0** 键测试加钱
