# Mod 开发快速开始

## 目录结构

```
res://mods/你的Mod名字/
├── mod.json                 # 必需：Mod 清单（英文默认）
├── mod_zh.json              # 推荐：中文名称/描述
├── scripts/
│   ├── main.gd              # 脚本入口（type: "script"）
│   └── ...                  # 所有 .gd 文件都会被加载
└── locale/
    └── zh-CN.json           # 可选：多语言翻译
```

## mod.json 格式

```json
{
  "id": "my_mod",
  "version": "1.0",
  "author": "你的名字",
  "type": "script",
  "dependencies": [],
  "optional_dependencies": [],
  "conflicts": [],
  "compatibility": {
    "game_version": ">=0.1"
  }
}
```

| 字段 | 说明 |
|------|------|
| `type` | `"script"`（GDScript）/ `"data"`（数据） |
| `id` | 唯一标识，用于 ModCommAPI 等跨 Mod 通信 |
| `dependencies` | 硬依赖，缺少则跳过加载 |
| `optional_dependencies` | 可选依赖，存在则排前面加载，不影响 |
| `conflicts` | 冲突列表，同时启用时报错 |

> `entry_point` 字段被游戏忽略。所有 `scripts/` 下的 `.gd` 文件都会被加载，其中定义了 `OnLoad(gm, bridge)` 的脚本会收到初始化调用。

### 名称与描述

`name` 和 `description` **不要**写在 `mod.json` 中。放入 `mod_{语言}.json`：

**mod_zh.json**（当前语言为中文时生效）
```json
{
  "name": "我的 Mod",
  "description": "功能介绍"
}
```

**mod_en.json**（当前语言为英文时生效）
```json
{
  "name": "My Mod",
  "description": "Feature description"
}
```

游戏按当前系统语言读取 `mod_{lang}.json`，找不到时回退到 `mod.json` 的 `name`/`description`。

## 脚本 Mod（推荐）

```gdscript
extends Node

var b: Variant = null

func OnLoad(gm: Node, bridge) -> void:
	# bridge 参数已弃用，通过单例访问
	b = Engine.get_singleton("ModBridge")
	if b == null:
		push_error("ModBridge 不可用")
		return
	b.log("Hello World!")

func OnUnload() -> void:
	b.log("Mod 已卸载")
```

- `gm` = GameManager（游戏主管理器）
- `Engine.get_singleton("ModBridge")` — 桥接单例，**所有 C# API 的蛇形封装**
- 所有 `scripts/` 下的 `.gd` 文件都会加载，只有定义 `OnLoad(gm, bridge)` 的脚本才会初始化
- `OnUnload()` 在 Mod 禁用时调用，用于释放快捷键、端点等资源
- 不要用 `_ready()`，游戏不会调用它

## 数据 Mod

在 `data/balance.json` 中覆盖游戏平衡参数，无需写代码：

```json
{
  "project.init.gameplay_base": 40,
  "fatigue.monthly_gain": 2,
  "sales.multiplier.95": 2.0
}
```

格式为**扁平键值对**（flat merge），使用点分隔路径直接对应游戏配置键。游戏使用 `TryLoadAndMerge` + `MergeFromJson` 将数据合并进配置表。

## 控制台命令

模组可以在控制台注册自定义命令：

```gdscript
func OnLoad(gm: Node, bridge) -> void:
	var b = Engine.get_singleton("ModBridge")
	b.register_console_command("mycmd", Callable(self, "_on_mycmd"))

func _on_mycmd(args: Array) -> void:
	b.log("执行 mycmd: " + str(args))
```

玩家在游戏中按 `~` 打开控制台，输入 `mycmd arg1 arg2` 即可调用。

## 工作示例

`mod_examples/` 目录下有可直接运行的示例：

| 示例 | 类型 | 功能 |
|------|------|------|
| `01_money_helper` | 脚本 | 每月加钱 + 按 0 键加钱 |
| `02_dev_booster` | 脚本 | F5 满分 / F6 清疲劳 / F7 全科技 |
| `03_balance_tweaker` | 数据 | 调整初始属性和销售参数 |

## 快速测试

1. 把 `mod_examples/01_money_helper` 复制到 `mods/` 下
2. 启动游戏 → 主菜单 → Mod → 勾选 Money Helper
3. 点"新游戏"进入游戏
4. 按 `~` 打开控制台，输入 `add_money 10000` 测试
