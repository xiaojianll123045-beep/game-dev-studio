# 自定义成就

添加自定义成就供玩家在游戏中解锁。

## 成就定义

在 `data/achievements/` 目录下创建成就文件：

```json
{
  "achievements": [
    {
      "id": "galactic_conqueror",
      "name": "银河征服者",
      "description": "征服所有敌对文明",
      "icon": "achievement_conqueror",
      "category": "conquest",
      "rarity": "legendary",
      "hidden": false,
      "steam_id": 100001,
      "requirements": {
        "type": "conquer_all_enemies",
        "count": 1,
        "params": {
          "difficulty_min": "normal"
        }
      },
      "rewards": [
        {"type": "title", "id": "conqueror"},
        {"type": "badge", "id": "golden_crown"},
        {"type": "resource", "resource": "prestige", "value": 500}
      ]
    }
  ]
}
```

## 成就字段

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `id` | String | 是 | 成就唯一标识 |
| `name` | String | 是 | 成就名称 |
| `description` | String | 是 | 成就描述 |
| `icon` | String | 否 | 图标名称 |
| `category` | String | 是 | 分类 |
| `rarity` | String | 否 | 稀有度 |
| `hidden` | Boolean | 否 | 是否隐藏（解锁前不可见） |
| `steam_id` | Integer | 否 | Steam 成就 ID |
| `requirements` | Object | 是 | 解锁条件 |
| `rewards` | Array | 否 | 解锁奖励 |

## 稀有度

| 稀有度 | 颜色 | 说明 |
|--------|------|------|
| `common` | 灰色 | 普通成就 |
| `rare` | 蓝色 | 稀有成就 |
| `epic` | 紫色 | 史诗成就 |
| `legendary` | 橙色 | 传说成就 |

## 条件类型

```json
{
  "requirements": {
    "type": "and",
    "conditions": [
      {
        "type": "stat_reach",
        "stat": "total_kills",
        "value": 1000
      },
      {
        "type": "have_component",
        "component_id": "legendary_weapon"
      },
      {
        "type": "win_game",
        "difficulty_min": "hard"
      }
    ]
  }
}
```

### 可用条件

| 条件类型 | 参数 | 说明 |
|---------|------|------|
| `stat_reach` | stat, value | 某项统计达到指定值 |
| `conquer_all_enemies` | difficulty_min | 征服所有敌人 |
| `have_component` | component_id | 拥有指定组件 |
| `win_game` | difficulty_min | 在指定难度下获胜 |
| `research_tech` | tech_id | 研发指定科技 |
| `collect_resources` | resource, value | 累积收集资源 |
| `complete_events` | event_id | 完成指定事件 |
| `fast_completion` | turns_max | 在指定回合内完成 |

## 奖励类型

| 奖励类型 | 参数 | 说明 |
|---------|------|------|
| `title` | id | 解锁称号 |
| `badge` | id | 解锁徽章 |
| `resource` | resource, value | 奖励资源 |
| `component` | id | 奖励组件 |
| `skin` | id | 解锁皮肤 |
| `ability` | id | 解锁能力 |

## 脚本集成

```gdscript
func _ready():
    bridge.register_hook("on_achievement_unlocked", self, "_on_achievement_unlocked")

func _on_achievement_unlocked(achievement_id, player_id):
    if achievement_id == "galactic_conqueror":
        bridge.log_info("恭喜解锁：银河征服者！")
        bridge.unlock_title(player_id, "conqueror")

    # 检查自定义成就条件
    var kills = bridge.get_player_stat(player_id, "total_kills")
    if kills >= 1000 and not bridge.is_achievement_unlocked(player_id, "war_veteran"):
        bridge.unlock_achievement(player_id, "war_veteran")
```

## 注意事项

- 成就 ID 必须是全局唯一的
- `steam_id` 需要与 Steamworks 配置一致
- 条件检查是自动进行的，无需手动编写检查逻辑
- 奖励在成就解锁时自动发放
