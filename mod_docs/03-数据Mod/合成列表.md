# 合成列表

定义资源合成与融合配方。

## 合成配方定义

在 `data/recipes/` 目录下创建配方文件：

```json
{
  "recipes": [
    {
      "id": "fusion_cell",
      "name": "聚变电池",
      "description": "将氢和氦融合为高能聚变电池",
      "category": "energy",
      "tier": 2,
      "inputs": [
        {"item": "hydrogen", "amount": 10},
        {"item": "helium", "amount": 5},
        {"item": "energy", "amount": 100}
      ],
      "output": {
        "item": "fusion_cell",
        "amount": 3
      },
      "byproduct": {
        "item": "waste_heat",
        "amount": 1,
        "chance": 0.3
      },
      "duration": 10.0,
      "facility": "fusion_lab",
      "requirements": {
        "tech": "fusion_power",
        "level": 1
      }
    }
  ]
}
```

## 配方字段

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `id` | String | 是 | 配方唯一标识 |
| `name` | String | 是 | 配方名称 |
| `description` | String | 否 | 配方描述 |
| `category` | String | 是 | 配方分类 |
| `tier` | Integer | 否 | 合成等级 |
| `inputs` | Array | 是 | 输入材料列表 |
| `output` | Object | 是 | 输出产物 |
| `byproduct` | Object | 否 | 副产物 |
| `duration` | Number | 否 | 合成时间 |
| `facility` | String | 否 | 所需设施 |
| `requirements` | Object | 否 | 前置条件 |

## 配方分类

| 分类 | 说明 |
|------|------|
| `basic` | 基础合成 |
| `energy` | 能源合成 |
| `materials` | 材料合成 |
| `components` | 组件合成 |
| `advanced` | 高级合成 |
| `special` | 特殊合成 |

## 输入与输出

```json
{
  "inputs": [
    {"item": "item_id", "amount": 数量, "quality_min": 最低品质},
    {"item_category": "category", "amount": 数量}
  ],
  "output": {
    "item": "item_id",
    "amount": 数量,
    "quality": "品质"
  },
  "byproduct": {
    "item": "item_id",
    "amount": 数量,
    "chance": 概率(0-1)
  },
  "consumable": true
}
```

## 连锁配方

```json
{
  "recipes": [
    {
      "id": "refine_titanium",
      "name": "精炼钛矿",
      "category": "materials",
      "inputs": [
        {"item": "raw_titanium", "amount": 5}
      ],
      "output": {
        "item": "titanium_ingot",
        "amount": 1
      },
      "duration": 5.0,
      "facility": "refinery"
    },
    {
      "id": "alloy_titanium_steel",
      "name": "钛钢合金",
      "category": "materials",
      "tier": 3,
      "inputs": [
        {"item": "titanium_ingot", "amount": 2},
        {"item": "steel_ingot", "amount": 3}
      ],
      "output": {
        "item": "titanium_steel",
        "amount": 1
      },
      "duration": 15.0,
      "facility": "advanced_factory",
      "requirements": {
        "tech": "alloy_metallurgy"
      }
    }
  ]
}
```

## 脚本集成

```gdscript
func _ready():
    bridge.register_hook("on_recipe_completed", self, "_on_recipe_completed")

func _on_recipe_completed(recipe_id, player_id, output_items):
    if recipe_id == "fusion_cell":
        bridge.log_info("聚变电池合成完成！")
        bridge.add_technology_progress(player_id, "fusion_power", 10)

    # 自定义合成逻辑
    bridge.process_recipe(player_id, recipe_id)
```

## 注意事项

- 输入中的 `item` 或 `item_category` 至少需要指定一个
- 副产物的 `chance` 字段控制生成概率
- `facility` 字段用于限制合成地点
- 连锁配方可以形成合成树，建议参考游戏内的配方链条
- 配方可通过语言包实现名称和描述的本地化
