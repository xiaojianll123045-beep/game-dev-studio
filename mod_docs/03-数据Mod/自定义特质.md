# 自定义特质

为文明、领袖或单位添加自定义特质。

## 特质定义

在 `data/traits/` 目录下创建特质文件：

```json
{
  "traits": [
    {
      "id": "industrial_giant",
      "name": "工业巨擘",
      "description": "工业产出大幅提升，但文化发展略微减缓",
      "type": "civilization",
      "tier": "major",
      "cost": 3,
      "icon": "trait_industry",
      "effects": [
        {
          "type": "modifier",
          "target": "production_output",
          "value": 1.25,
          "operation": "multiply"
        },
        {
          "type": "modifier",
          "target": "culture_output",
          "value": 0.85,
          "operation": "multiply"
        }
      ],
      "requirements": {
        "min_population": 10
      },
      "conflicts": ["agrarian_society", "trade_empire"],
      "tags": ["industry", "production"]
    }
  ]
}
```

## 特质字段

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `id` | String | 是 | 特质唯一标识 |
| `name` | String | 是 | 特质名称 |
| `description` | String | 是 | 特质描述 |
| `type` | String | 是 | 特质类型 |
| `tier` | String | 否 | 等级（`minor`/`major`/`unique`） |
| `cost` | Integer | 否 | 选择消耗 |
| `icon` | String | 否 | 图标 |
| `effects` | Array | 是 | 效果列表 |
| `requirements` | Object | 否 | 前置条件 |
| `conflicts` | Array | 否 | 冲突特质列表 |
| `tags` | Array | 否 | 标签 |

## 特质类型

| 类型 | 应用对象 | 说明 |
|------|---------|------|
| `civilization` | 文明 | 选择文明时选取 |
| `leader` | 领袖 | 领袖特有特质 |
| `unit` | 单位 | 单位特有特质 |
| `population` | 人口 | 人口特质 |

## 效果系统

```json
{
  "effects": [
    {
      "type": "modifier",
      "target": "生产/属性路径",
      "value": 数值,
      "operation": "add" | "multiply"
    },
    {
      "type": "unlock",
      "target": "unlock_type",
      "id": "解锁ID"
    },
    {
      "type": "ability",
      "id": "能力ID",
      "cooldown": 60.0
    },
    {
      "type": "immunity",
      "target": "负面效果ID"
    },
    {
      "type": "bonus_vs",
      "target": "目标类型",
      "value": 0.5
    }
  ]
}
```

## 脚本集成

```gdscript
func _ready():
    bridge.register_hook("on_trait_applied", self, "_on_trait_applied")
    bridge.register_hook("on_trait_removed", self, "_on_trait_removed")

func _on_trait_applied(trait_id, target_id, target_type):
    if trait_id == "industrial_giant" and target_type == "civilization":
        bridge.log_info("工业巨擘特质已应用")
        bridge.add_special_building(target_id, "auto_factory")

func _on_trait_removed(trait_id, target_id, target_type):
    if trait_id == "industrial_giant":
        bridge.log_info("工业巨擘特质已移除")
        bridge.remove_special_building(target_id, "auto_factory")
```

## 冲突系统

特质可以通过 `conflicts` 字段声明互斥关系：

```json
{
  "conflicts": ["agrarian_society", "trade_empire"]
}
```

冲突特质不能同时选择。冲突检查是双向的——如果 A 冲突 B，则 B 也会自动冲突 A。

## 注意事项

- 主要特质（`major`）通常提供显著增益同时带有负面效果
- 消耗点数 `cost` 用于特质选择系统，总点数有限
- 冲突列表用于防止效果叠加不平衡
- 特质效果可通过脚本进一步扩展
