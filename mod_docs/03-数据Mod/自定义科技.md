# 自定义科技

通过数据模组添加全新的科技树条目。

## 科技定义

在 `data/techs/` 目录下创建科技文件：

```json
{
  "techs": [
    {
      "id": "neural_interface",
      "name": "神经接口",
      "description": "开发脑机接口技术，提升单位响应速度",
      "category": "engineering",
      "tier": 3,
      "cost": 500,
      "research_time": 120.0,
      "prerequisites": ["cybernetics", "advanced_computing"],
      "unlocks": [
        {
          "type": "component",
          "id": "neural_link"
        },
        {
          "type": "bonus",
          "target": "unit_response_time",
          "value": -0.2
        },
        {
          "type": "building",
          "id": "neural_lab"
        }
      ],
      "flavor_text": "当思维与机器融为一体，战争的形态将彻底改变。"
    }
  ]
}
```

## 科技字段

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| `id` | String | 是 | 科技唯一标识 |
| `name` | String | 是 | 科技名称 |
| `description` | String | 是 | 科技描述 |
| `category` | String | 是 | 科技类别 |
| `tier` | Integer | 是 | 科技层级 |
| `cost` | Number | 是 | 研究消耗 |
| `research_time` | Number | 是 | 研究时间（秒） |
| `prerequisites` | Array | 否 | 前置科技列表 |
| `unlocks` | Array | 是 | 解锁内容 |
| `flavor_text` | String | 否 | 背景故事文本 |

## 科技类别

| 类别 | 描述 |
|------|------|
| `engineering` | 工程学 |
| `physics` | 物理学 |
| `biology` | 生物学 |
| `military` | 军事科技 |
| `economy` | 经济学 |
| `special` | 特殊科技 |

## 解锁类型

`unlocks` 数组支持多种解锁类型：

```json
{
  "unlocks": [
    {"type": "component", "id": "组件ID"},
    {"type": "building", "id": "建筑ID"},
    {"type": "unit", "id": "单位ID"},
    {"type": "ability", "id": "技能ID"},
    {"type": "bonus", "target": "属性路径", "value": 数值},
    {"type": "recipe", "id": "配方ID"},
    {"type": "feature", "id": "功能ID"}
  ]
}
```

## 科技树分支

```json
{
  "techs": [
    {
      "id": "basic_shield",
      "name": "基础护盾",
      "category": "engineering",
      "tier": 1,
      "cost": 100,
      "prerequisites": [],
      "unlocks": [
        {"type": "component", "id": "shield_mk1"}
      ]
    },
    {
      "id": "advanced_shield",
      "name": "高级护盾",
      "category": "engineering",
      "tier": 2,
      "cost": 300,
      "prerequisites": ["basic_shield"],
      "unlocks": [
        {"type": "component", "id": "shield_mk2"}
      ]
    },
    {
      "id": "plasma_shield",
      "name": "等离子护盾",
      "category": "engineering",
      "tier": 3,
      "cost": 800,
      "prerequisites": ["advanced_shield", "plasma_physics"],
      "unlocks": [
        {"type": "component", "id": "shield_mk3"}
      ]
    }
  ]
}
```

## 脚本集成

```gdscript
func _ready():
    bridge.register_hook("on_tech_researched", self, "_on_tech_researched")

func _on_tech_researched(tech_id, player_id):
    if tech_id == "neural_interface":
        bridge.log_info("神经接口已研发完成！")
        bridge.apply_modifier(player_id, "unit_response_speed", -0.2)
```

## 注意事项

- 确保前置科技 ID 正确且不造成循环依赖
- 科技成本应参考同层级科技保持平衡
- `tier` 决定科技树中的纵向位置
- 可在语言包中为 `name` 和 `description` 使用键值引用
