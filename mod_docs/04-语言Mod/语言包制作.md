# 语言包制作

语言模组用于翻译游戏文本或添加新的语言支持。

## 文件结构

语言包文件放在 `locale/` 目录下，以语言代码命名：

```
locale/
├── zh-CN.json       # 简体中文
├── zh-TW.json       # 繁体中文
├── en.json          # 英文
├── ja.json          # 日文
└── ko.json          # 韩文
```

## mod.json 配置

```json
{
  "id": "chinese_lang_pack",
  "name": "中文语言包",
  "version": "1.0.0",
  "author": "Translator",
  "description": "提供完整的中文翻译",
  "type": "language",
  "compatibility": {
    "game_version": ">=1.0.0"
  }
}
```

## JSON 格式

语言包使用键值对格式：

```json
{
  "locale": "zh-CN",
  "strings": {
    "ui.main_menu.title": "银河帝国",
    "ui.main_menu.new_game": "新游戏",
    "ui.main_menu.load_game": "加载游戏",
    "ui.main_menu.settings": "设置",
    "ui.main_menu.quit": "退出游戏",
    "ui.main_menu.mods": "模组",
    "ui.settings.audio": "音频设置",
    "ui.settings.video": "视频设置",
    "ui.settings.controls": "控制设置",
    "ui.settings.language": "语言",
    "ui.loading.tip": "小提示：点击右键可以快速取消选中",
    "ui.notification.research_complete": "研究完成：{tech_name}",
    "ui.notification.build_complete": "建造完成：{building_name}",
    "ui.notification.unit_lost": "失去单位：{unit_name}",
    "ui.dialog.confirm": "确认",
    "ui.dialog.cancel": "取消",
    "ui.dialog.yes": "是",
    "ui.dialog.no": "否"
  }
}
```

## 数据模组的多语言支持

如果你创建了数据模组，可以同时提供语言包为新内容提供翻译：

```json
{
  "strings": {
    "component.reactor_mk3.name": "反应堆 Mk3",
    "component.reactor_mk3.desc": "高级核聚变反应堆，提供大量能量输出",
    "tech.neural_interface.name": "神经接口",
    "tech.neural_interface.desc": "开发脑机接口技术，提升单位响应速度",
    "event.asteroid_strike.name": "小行星撞击",
    "event.asteroid_strike.desc": "一颗巨型小行星正朝星球袭来！",
    "achievement.galactic_conqueror.name": "银河征服者",
    "achievement.galactic_conqueror.desc": "征服所有敌对文明",
    "trait.industrial_giant.name": "工业巨擘",
    "trait.industrial_giant.desc": "工业产出大幅提升，但文化发展略微减缓"
  }
}
```

## 键值命名规范

推荐使用点分命名法：

```
<域>.<子域>.<具体项>.<属性>
```

| 域 | 用途 |
|----|------|
| `ui` | 界面文本 |
| `component` | 组件名称和描述 |
| `tech` | 科技名称和描述 |
| `event` | 事件文本 |
| `achievement` | 成就文本 |
| `trait` | 特质文本 |
| `unit` | 单位文本 |
| `building` | 建筑文本 |
| `notification` | 通知消息 |
| `dialog` | 对话框文本 |
| `tooltip` | 工具提示 |
| `misc` | 杂项文本 |

## 变量替换

文本中可以使用 `{变量名}` 进行动态替换：

```json
{
  "strings": {
    "notification.attack": "{attacker} 正在攻击 {defender}！",
    "notification.trade": "与 {other_empire} 的贸易已完成，获得 {amount} 信用点",
    "ui.tooltip.damage": "攻击力：{damage}\n防御力：{defense}\n射程：{range}"
  }
}
```

## 复数与条件格式

```json
{
  "strings": {
    "ui.population.count": "{count} 人口",
    "ui.turns.remaining": "剩余 {turns} 回合",
    "ui.resource.tooltip": "{name}: {current}/{max}\n收入: {income}/回合"
  }
}
```

## 脚本中使用本地化字符串

```gdscript
func _ready():
    # 获取本地化文本
    var text = bridge.tr("ui.notification.research_complete", {"tech_name": "核聚变"})
    bridge.log_info(text)

    # 注册动态字符串
    bridge.register_dynamic_string("mod.custom_greeting", "欢迎使用自定义模组！")

    # 显示通知
    bridge.show_notification("mod.custom_greeting")
```

## 注意事项

- 语言包文件编码必须为 UTF-8（无 BOM）
- 确保所有键值在游戏原文中都有对应的引用
- 同一语言的多个语言模组会合并，后加载的覆盖先加载的
- 建议为数据模组中新增的内容提供翻译，否则会显示键值原文
- 测试时切换到对应语言以验证翻译是否正确显示
