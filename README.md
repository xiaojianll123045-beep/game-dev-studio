# 游戏开发商 / Game Developer Studio

一个以独立游戏开发为主题的经营模拟/沙盒策略游戏。
An indie game dev business sim / sandbox strategy game.

Built with Godot 4.7 (Mono / C# .NET 8).

### 许可证 / License

游戏核心代码以 **GNU General Public License v3 (GPL v3)** 发布。

**Mod 例外：** 为本游戏制作的 Mod（脚本、数据包、资源包、插件等）**不需要开源**，可以用任何许可证发布。

**沙箱例外：** 沙箱系统（`ModSandbox.cs` 及 `extensions/mod_sandbox_native/`）可被任何人单独拿走使用，**无需开源你的项目**。唯一要求是在用户可见的位置（关于页/致谢）署名并附上本仓库地址。详见 `LICENSE`。

## Mod 开发 / Mod Development

Mod 系统已就绪，支持：
- **数据 Mod** — 修改数值、配方、事件等
- **语言 Mod** — 添加/修改翻译
- **脚本 Mod** — 用 Godot GDScript 写逻辑扩展
- **Mod API** — `ModAPI` / `ModBridge` 提供丰富的交互接口

详细文档见 `mod_docs/` 目录。
Mod examples 见 `mod_examples/` 目录。
