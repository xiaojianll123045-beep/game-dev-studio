# 游戏开发商 / Game Developer Studio

一个以独立游戏开发为主题的经营模拟/沙盒策略游戏。
An indie game dev business sim / sandbox strategy game.

Built with Godot 4.7 (Mono / C# .NET 8).

## Mod 开发 / Mod Development

Mod 系统已就绪，支持：
- **数据 Mod** — 修改数值、配方、事件等
- **语言 Mod** — 添加/修改翻译
- **脚本 Mod** — 用 Godot GDScript 写逻辑扩展
- **Mod API** — `ModAPI` / `ModBridge` 提供丰富的交互接口

详细文档见 `mod_docs/` 目录。
Mod examples 见 `mod_examples/` 目录。
